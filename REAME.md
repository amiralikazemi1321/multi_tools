# 🚀 Ultra V4 Tool

A powerful multi-purpose desktop application built with **Python + PyQt5**.

Amirali Ultra V4 is a personal productivity toolkit that combines useful tools in one modern desktop app.

## ✨ Features

### 🧠 AI Chat
- Chat with AI models using OpenAI-compatible API
- Supports custom Base URL
- Supports multiple models
- API configuration saved locally

### 📝 Notes Manager
- Create notes
- Edit notes
- Save automatically
- Local JSON storage

### ✅ Todo List
- Add tasks
- Manage your daily tasks
- Persistent storage

### 🎮 Mini Game
- Number guessing game
- Simple built-in entertainment

### 🌙 Modern UI
- Dark theme
- PyQt5 interface
- Multi-page navigation
- Home / Back navigation system

---

## 🛠️ Technologies

- Python 3
- PyQt5
- OpenAI Python SDK
- JSON storage
